## Best Practices for AI-Generated Screensavers

### Image Variety
- **Different prompts for each image** - Don't reuse the same prompt
- Vary: time of day, weather, angle, interior style
- Example: same cabin theme but - morning light, sunset, night, snowfall, rain

### Transition Techniques
- Crossfade between images (3-5 second dwell)
- Ken Burns effect (slow zoom/pan)
- Match color tones for smooth transitions

### Video Settings
- 10-15 minute loop ideal
- 24-30 fps for smooth playback
- 1920x1080 or 3840x2160 for HD/4K
